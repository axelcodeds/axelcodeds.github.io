Featured Projects

AURA Guerrero

AURA Guerrero is an open-source web application that provides public access to air quality and UV radiation data for Guerrero, Mexico. It displays this information on an interactive municipal map, helping citizens, authorities, and tourists make informed health decisions. The project is a response to a NASA challenge and aims to establish an early warning system for environmental risks.
